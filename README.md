# nophenia-Simple-Debug-Menu
### Simple Debug Menu for nophenia, is (probably) NOT compatible with multiplayer at the moment.

# Install
- Install Python 3 if you don't already have it.
- Download nophenia-debug-menu-v1.0.0.zip (Windows) or nophenia-debug-menu-v1.0.0.tar.gz (Linux) and extract it.
- Open a terminal in the extracted folder and run install.py with the path to your nophenia game folder (the one containing the game executable):
#### - Linux: python3 install.py "/path/to/steamapps/common/nophenia"
#### - Windows: python install.py "C:\Program Files (x86)\Steam\steamapps\common\nophenia" (run the terminal as Administrator if the game is in Program Files)
Launch the game. Debug Menu is available in the phone menu or through keybinds listed below.

# Uninstall
(To uninstall: python install.py --uninstall "<game folder>")

## KEYBINDS:
- F2 Debug Control Panel (More friendly Debug Menu UI in the phone)
- F3 No-clip (Fly, but you can phase through anything)
- F4 Fly (Self explanatory) 
- F5 Freecam (No-clipping camera)
- F6 Spawn Menu (Allows you to spawn-in full maps on top of maps and etc. VERY BROKEN)
- F7 Unstuck (Teleports you up slightly just in the scenario you get stuck)
- F8, left and right arrow key + shift, Page up, Page down Flashlight calibration (Only available with this mod paired with the Sightless Lite mod and using the F2 Debug Menu)
- +, - No-clip speed control

### WARNING: Spawn menu is VERY broken and WILL break your game. Also, Debug Menu paired with the Flashlight mod allows flashlight calibration

## Suggestions/Tips:
- This mod is intended for exploration/having fun once finished with the actual game.
- The Debug Menu in the phone is easier and less buggy to use than the keybinds.
- Flashlight calibration is only available with the Sightless Lite mod paired with this Debug Menu mod and is only able to be calibrated with the F2 Debug Menu, but the default position should already be fine.
