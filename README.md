# nophenia-Simple-Debug-Menu
### Simple Debug Menu made for nophenia, not officially compatible with multiplayer at the moment, but technically does work on multiplayer with some slight tweaking.

## Install
Install Python 3 - get it from https://python.org (on Windows, tick "Add Python to PATH" during setup).

Download nophenia-debug-menu-v1.0.0.zip (Windows) or nophenia-debug-menu-v1.0.0.tar.gz (Linux) and extract it.

Open the extracted folder and:

Windows: double-click install.bat
Linux: run install.sh (or python3 install.py in a terminal)
It finds your nophenia folder on its own (or asks you to paste/drag it in). No typing paths needed.

Launch the game. F2 opens the debug overlay, debug menu also exists on the cellphone in-game.

(To remove it, run the installer again and add --uninstall.)

Stuck/"syntax error" or another error? Just double-click install.bat (Windows, administator might be required sometimes) or run install.sh (Linux).

## KEYBINDS/FEATURES:
- F2 Debug Control Panel (Cordinates, scene, SL-cali, etc. More UI friendly Debug Menu UI in the phone)
- F3 No-clip (Fly, but you can phase through anything)
- F4 Fly (Self explanatory) 
- F5 Freecam (No-clipping camera)
- F6 Spawn Menu (Allows you to spawn-in full maps on top of maps and etc. VERY BROKEN)
- F7 Unstuck (Teleports you up slightly just in the scenario you get stuck)
- F8, left and right arrow key + shift, Page up, Page down Flashlight calibration (Only available with this mod paired with the Sightless Lite mod and using the F2 Debug Menu)
- +, - No-clip speed control

### WARNING: Spawn menu is VERY broken and WILL probably break your game.

## Suggestions/Tips:
- This mod is intended for exploration/having fun once finished with the actual game.
- The Debug Menu in the phone is easier and less buggy to use than the keybinds.
- Flashlight calibration is only available with the Sightless Lite mod paired with this Debug Menu mod and is only able to be calibrated with the F2 Debug Menu, but the default position should already be fine for most people.
